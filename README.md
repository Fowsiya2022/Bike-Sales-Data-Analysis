# Bike Sales Data Analysis

This project analyzes customer demographic data to understand factors influencing bike purchases.

The dataset contains over 1000 customer records including information about income, age, occupation, region, and commute distance.

The analysis was performed using Microsoft Excel with pivot tables and visualizations.

## Key Insights

- Customers with higher income are more likely to purchase bikes
- The Pacific region shows the highest bike purchase rate
- Younger adults show slightly higher purchasing behaviour
- Gender does not significantly affect bike purchases

## Tools Used

- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Data Visualization
